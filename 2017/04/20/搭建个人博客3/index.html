第一次写博客,还有点小激动呢.这个个人博客是看了好几位大神的博客才算搭建好的,下面是他们的链接.

使用Hexo建立独立博客

Next主题小修改

以及hexo的官网Hexo

安装之前

首先安装nodejs,在官网下载安装.

安装hexo

使用hexo

创建博客文件夹

初始化已经完成,使用hexo s预览

常用命令

hexo new [文章标题] ##会在source/posts目录下生成新的空白文章

hexo clean ##清除缓存

hexo s ##启动本地服务器

hexo g ##生成网站

hexo d ##推送发布文章

连接github

创建账号就不多说了.

创建”用户名.github.io”仓库

创建一个”你的用户名.github.io”的仓库,以后查看博客的网址为https://用户名.github.io.

添加ssh密钥

在命令行执行

其中会有三次确认,第一次是sshkey的保存位置,第二次是创建一个密码,第三次确认密码.

之后将复制好的密钥粘贴在github>设置>ssh里就好了.

详细步骤请看官网Generating a new SSH key

测试连接:

过程中会要求输入账号密码,成功之后会显示自己的用户名.

配置hexo

打开_config.yml文件,修改deploy模块

之后执行 hexo d 就可以了.

如果deploy报错

可以试试安装hexo-deploy-git
